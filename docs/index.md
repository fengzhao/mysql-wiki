# MySQL WIKI

## MySQL版本

本站点中的示例代码主要针对 **MySQL** `8.0` 版本，部分内容适用于 `5.7` 版本。

## 快速开始

如果你是 **初学者**，建议从 **[MySQL入门](/basic/0.mysql_source/)** 开始系统学习 MySQL 基础知识。

如果你已经掌握基础知识，想深入了解 **MySQL** 的内部机制，可以阅读 **[MySQL高级](/advanced/深入理解innoDB存储引擎/)** 部分。

如果你关注 **MySQL性能优化**，可以查看 **[MySQL调优](/MySQL调优/)** 章节。

如果你想了解其他数据库，我们还提供了 **[PostgreSQL](/PostgreSQL/)**、**[MongoDB](/MongoDB/)**、**[Redis](/Redis/)** 等数据库的学习资料。

如果你发现文章 **有错误**，请在 GitHub 上提交 Issue 或 Pull Request 帮助修正。
